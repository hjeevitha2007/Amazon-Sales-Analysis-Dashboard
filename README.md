# Amazon-Sales-Analysis-Dashboard
Built an interactive sales dashboard using Excel and Power BI. Analyzed 120K+ orders, identified top-performing categories and states, and created visualizations to support business decision-making.
# Amazon Sales Analysis Dashboard

## Project Overview
Analyzed Amazon sales data using Excel and Power BI.

## Tools Used
- Excel
- Power BI

## Key Metrics
- Total Revenue: ₹78.59M
- Total Orders: 120K+
- Average Order Value: ₹609.36

## Insights
- Set category generated the highest revenue.
- Maharashtra was the top-performing state.
- Sales trends were analyzed monthly.
